当前版本：云满客客户管理系统V1.1

更新时间：2015年10月25日

功能如下：

一、客户资料管理

    1.用户可通过账号、邮箱登陆系统；
    2.自定义客户性质、来源、区域、行业；
    3.自定义客户类型,可通过客户类型查询客户；
    4.录入客户相关资料，可随跟踪客户；
    5.客户的业务列表（最新十条）；
    6.客户随时删除，清除过期客户；

二、业务管理
    1.录入客户的相关业务，可多条，并且可以设置业务起始、结束时间、成本价格、销售价格等；
    2.客户业务的修改、删除等；
    3.客户业务的查询：按照客户名称、到期时间、业务名称模糊查询；

服务器（空间）环境要求，支持PHP+MYSQL，最好使用Linux环境，支持伪静态(目前已经关闭)。

直接将根目录yunmanke.sql导入到msyql数据库中。默认建立一个数据库为“yunmanke”的数据库，然后导入。
将文件拷贝到根目录。

修改数据库配置，文件夹地址：\app\config\database.php

$db['default']['hostname'] = 'localhost'; //数据库地址。本地为localhost
$db['default']['username'] = 'root';      //数据库用户名
$db['default']['password'] = '1';    //数据库密码
$db['default']['database'] = 'yunmanke'; //数据库名称
